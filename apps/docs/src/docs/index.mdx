## Quick Start

Create a virtual account with a single API call:

```bash
curl -X POST https://api.paygrid.xyz/v1/accounts/virtual \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "accountRef": "cust-abc123",
    "accountName": "Acme Inc"
  }'
```

The response includes a unique bank account number your customer can use to receive payments:

```json
{
  "code": "00",
  "description": "Success",
  "data": {
    "accountRef": "cust-abc123",
    "accountName": "Acme Inc",
    "bankName": "Nombank MFB",
    "bankAccountNumber": "9391076543",
    "currency": "NGN"
  }
}
```
