# PayGrid Documentation

Welcome to the PayGrid developer documentation. PayGrid is a payment management dashboard powered by Nomba's API.

## Overview

PayGrid provides a simple interface to manage virtual accounts, track transactions, handle webhooks, and verify business KYC tiers.
