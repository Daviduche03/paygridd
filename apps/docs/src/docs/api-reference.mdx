# API Reference

## Authentication

**Issue Token**

```bash
POST /v1/auth/token/issue
```

Obtain an access token using client credentials.

**Refresh Token**

```bash
POST /v1/auth/token/refresh
```

Refresh an expiring access token.

## Virtual Accounts

**Create Virtual Account**

```bash
POST /v1/accounts/virtual
```

Creates a static virtual account for receiving payments.

**Create for Sub-Account**

```bash
POST /v1/accounts/virtual/{subAccountId}
```

Creates a virtual account linked to a sub-account.

**Get Virtual Account**

```bash
GET /v1/accounts/virtual/{identifier}
```

Retrieve details by account number or reference.

**List Virtual Accounts**

```bash
GET /v1/accounts/virtual/list
```

List virtual accounts with optional filters.

**Update Virtual Account**

```bash
PATCH /v1/accounts/virtual/{identifier}
```

Update account name, callback URL, or expected amount.

**Expire Virtual Account**

```bash
DELETE /v1/accounts/virtual/{identifier}
```

Expire a virtual account.

## Transactions

**List Parent Account Transactions**

```bash
GET /v1/transactions/account
```

**List Virtual Account Transactions**

```bash
GET /v1/accounts/virtual/{identifier}/transactions
```

## Transfers

**Bank Transfer**

```bash
POST /v1/transfers/bank
```

Transfer funds to a bank account.
