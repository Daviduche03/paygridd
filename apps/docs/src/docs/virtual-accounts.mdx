# Virtual Accounts

Generate unique NGN account numbers for your customers to receive payments.

## Account Types

| Type | Description |
|------|-------------|
| **Static** | Permanent account number for recurring customers |
| **Dynamic** | Temporary account with optional expiry and expected amount |

## Create a Virtual Account

Creates a static virtual account linked to your main account.

```bash
curl -X POST https://api.paygrid.xyz/v1/accounts/virtual \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "accountRef": "cust-abc123",
    "accountName": "John Doe"
  }'
```

### Response

```json
{
  "code": "00",
  "description": "Success",
  "data": {
    "accountRef": "cust-abc123",
    "accountName": "John Doe",
    "bankName": "Nombank MFB",
    "bankAccountNumber": "9391076543",
    "currency": "NGN"
  }
}
```

## Create for a Sub-Account

Creates a virtual account linked to a specific sub-account:

```bash
curl -X POST https://api.paygrid.xyz/v1/accounts/virtual/{subAccountId} \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "accountRef": "cust-abc123",
    "accountName": "John Doe"
  }'
```

## Update a Virtual Account

```bash
curl -X PATCH https://api.paygrid.xyz/v1/accounts/virtual/{identifier} \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "accountName": "Jane Doe",
    "callbackUrl": "https://your-app.com/webhooks/callback"
  }'
```
