# Webhooks

Nomba sends HTTP POST requests to your webhook URL when events occur on your account.

## Supported Events

| Event | Description |
|-------|-------------|
| `payment_success` | Payment credited to your Nomba account |
| `payout_success` | Payout debited from your account |
| `payment_failed` | Payment attempt failed |
| `payment_reversal` | Payment reversed back to customer |
| `payout_failed` | Payout failed to process |
| `payout_refund` | Payout refunded to your account |

## Configuration

Set your webhook URL in the Nomba dashboard under **Developer > Webhook Setup**.

Your webhook URL: `https://paygrid.xyz/webhooks/nomba`

## Signature Verification

Every webhook includes `nomba-signature` and `nomba-timestamp` headers. Verify the HMAC-SHA256 signature using your secret key before processing.
